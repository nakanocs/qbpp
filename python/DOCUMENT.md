# PyQUBO++ Document
This documentation for PyQUBO++ (Python binding of QUBO++) is currently under development.
Some pages may contain incomplete or provisional information.

## Installation
PyQUBO++ can be installed via pip:
```bash
pip install pyqbpp
```
